# 📄 Markdown to PDF Editor & Previewer

A modern, fast, and developer-friendly **Markdown editor & previewer** that lets you write, visualize, and export your `.md` files effortlessly.

---

## ✨ Features

* 📝 **Live Markdown Preview**
  Instantly renders `.md` files as you write or upload them.

* 🎨 **Syntax Highlighting**
  Powered by **Highlight.js** for beautiful, VS Code–style code blocks.

* 🧠 **Diagram & Mindmap Support**
  Create diagrams and flowcharts using **Mermaid**.

* 📤 **Export Options**

  * Export as **PDF**
  * Download as **Markdown (.md)**

* 🌗 **Theme Toggler**
  Switch between light and dark modes for a comfortable writing experience.

* ⚡ **Fast & Lightweight**
  Built with performance and simplicity in mind.

---

## 🛠️ Tech Stack

* **Marked.js** → Parses Markdown into HTML
* **Highlight.js** → Syntax highlighting for code blocks
* **Mermaid.js** → Diagram and visualization support

---

## 🚀 How It Works

1. Write a markdown `.md` file
2. The editor parses it using **Marked.js**
3. Code blocks are styled using **Highlight.js**
4. Diagrams are rendered via **Mermaid**
5. Preview updates in real-time
6. Export your work as PDF or Markdown anytime

---

## 📂 Supported Content

* Headings, lists, tables
* Code blocks with syntax highlighting
* Mermaid diagrams (flowcharts, sequence diagrams, mindmaps)
* Inline formatting (bold, italic, links, etc.)

---

## 📸 Preview

> Real-time preview ensures what you see is what you export.

---

## 💡 Usage

* Open the app in your browser
* Start writing Markdown `.md` file
* Toggle themes as needed
* Export when you're done
